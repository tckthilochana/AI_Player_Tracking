# Player Detection and Tracking using YOLOv8

## Objective
Detect and track players in football, cricket and rugby videos using YOLOv8.

## Models Used
- YOLOv8n
- YOLOv8n-pose

## Features
- Player Detection
- Player Tracking
- Pose Estimation

## Dataset
6 Sports Videos
- Football ×2
- Cricket ×2
- Rugby ×2

## Requirements
ultralytics
opencv-python
numpy

## Run

pip install ultralytics

python PlayerTracking.ipynb
